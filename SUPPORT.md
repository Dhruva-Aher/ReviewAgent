# Support

Welcome to the PRBeliefs (ReviewAgent) Support page! If you are experiencing issues, have questions, or want to request a feature, you're in the right place.

## Project Overview

PRBeliefs is an autonomous PR review system that persists team conventions and applies them to future pull requests. It uses a multi-agent orchestration architecture to review security, architecture, performance, dependencies, and test coverage.

## How to Report Bugs

If you encounter a bug or unexpected behavior, please check our [GitHub Issues](https://github.com/Dhruva-Aher/ReviewAgent/issues) to see if it has already been reported.

If it hasn't, please open a new issue with the following details:
1. A clear and descriptive title.
2. Steps to reproduce the issue.
3. Expected vs. actual behavior.
4. Relevant logs or error messages.

## Feature Requests

We love hearing your ideas! To request a new feature:
1. Open a new issue in our [GitHub Issues](https://github.com/Dhruva-Aher/ReviewAgent/issues).
2. Use the "Feature Request" label (if available) or simply prefix your title with `[Feature]`.
3. Describe the feature, the problem it solves, and how it would benefit your workflow.
