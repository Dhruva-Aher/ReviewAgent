# Privacy Policy

PRBeliefs is a GitHub App that reviews pull requests.

## Data we access
- Pull request diffs (read only)
- Repository metadata (name, owner)

## Data we store
- Team rules and past decisions (beliefs.json, stored locally on your server)
- No personal data is collected or stored

## Data we share
- We do not sell or share any data with third parties
